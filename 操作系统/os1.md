## 1.1 操作系统的作用、目标

### 1.1.1 操作系统作用

操作系统是系统软件——从计算机系统组成观点

​       计算机系统组成

```
软件：
应用软件
系统软件（操作系统、语言处理系统和常见的例行服务程序）

硬件：   
输入/输出（I/O）设备
存储器
处理器
```



操作系统是资源管理者——从资源管理观点，用于控制和管理计算机系统的硬件和软件资源

​       

```
计算机系统资源：
软件资源：信息（数据和程序）
硬件资源：I/O设备、存储器、处理器

操作系统管理：文件系统、设备管理、存储器管理、处理器（进程）管理
```

操作系统是扩充裸机的第一层系统软件——从软件分层、扩允机器
